# kocsistibor

Pénzügyi tanácsadás weboldala — Kocsis Tibor ("Tibi"). Magyar nyelvű, Astro 6 + Tailwind CSS v4 (AstroWind
sablon alapján).

## Dev

```bash
npm install
npm run dev
npm run build
```

## Deploy (Cloudflare)

```bash
npm run deploy   # astro build && wrangler deploy
```

Lásd [CLAUDE.md](./CLAUDE.md) a projekt kontextusához és az élesítés előtti TODO-listához (MNB engedélyszám,
valós ÁSZF/adatvédelmi szöveg, Web3Forms kulcs, telefonszám, fotó, statisztikák).
